# Bank Marketing Prediction

## Project Overview

This project predicts whether a customer will subscribe to a bank term deposit based on demographic and financial information.

## Objectives

* Analyze customer data.
* Preprocess and clean the dataset.
* Handle class imbalance.
* Train and compare multiple machine learning models.
* Predict customer responses to marketing campaigns.

## Techniques Used

* Data Cleaning
* Missing Value Handling
* Outlier Detection
* Feature Encoding
* Feature Selection
* SMOTE
* Model Evaluation

## Models

* Logistic Regression
* Random Forest
* XGBoost

## Results

XGBoost achieved the best performance with approximately 92.9% accuracy.

## Technologies

Python, Pandas, NumPy, Scikit-learn, XGBoost, Matplotlib, Seaborn
